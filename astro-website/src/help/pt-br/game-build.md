# Construa o prompt que você recebeu

Construa como você interpreta o prompt. Não desvie dos jogos de propósito!

- Para pular, use `/skip`.
- Veja comandos relacionados à construção usando `/help`.
- Use `/report` se o prompt for inapropriado.
