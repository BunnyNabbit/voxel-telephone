# /fastforward

Uso: `/fastforward <ações>`

Pode ser usado se o VCR estiver ativado com `/vcr`.

Restaura um número de ações revertidas pelo `/rewind`. Se nenhum argumento for fornecido, apenas uma ação será restaurada.

Aliases: `/ff`, `/redo`
