# /rewind

Uso: `/rewind <ações>`

Pode ser usado se o VCR estiver ativado com `/vcr`.

Reverte um número de ações realizadas em um nível. Se nenhum argumento for fornecido, apenas uma ação será revertida.

Aliases: `/rw` `/undo`
