# /fastforward

Uso: `/fastforward <actions>`

Utilizable si VCR está habilitado actualmente con `/vcr`.

Restaura un número de acciones revertidas por `/rewind`. Si no se da ningún argumento, sólo se restaurará una acción.

Aliases: `/ff`, `/redo`
