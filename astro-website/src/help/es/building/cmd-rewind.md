# /rewind

Uso: `/rewind <actions>`

Utilizable si VCR está habilitado actualmente con `/vcr`.

Invierte una serie de acciones realizadas a un nivel. Si no se da ningún argumento, sólo una acción será revertida

Aliases: `/rw` `/undo`
